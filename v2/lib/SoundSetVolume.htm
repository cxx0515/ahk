SoundSetVolume

更改声音设备的音量设置.

SoundSetVolume NewSetting , Component, Device

参数

NewSetting

类型: 字符串, 整数或浮点数

一个包含 -100 和 100(包含的) 之间的百分数的字符串. 如果这个数字以加号或减号开头, 则将 当前设置 往上或往下调指示的数量. 否则, 此设置将被明确地设置为 NewSetting 表示的级别.

如果百分数以减号开头或无符号开头, 则不需要用引号括起来.

Component

类型: 字符串或整数

如果为空或省略, 则更改主音量设置. 否则, 指定组件的显示名称和/或索引. 例如, 1, "Line in" 或 "Line in:2".

有关详情, 请参阅 Component (Sound 函数).

Device

类型: 字符串或整数

如果为空或省略, 则它默认为系统的默认回放设备(不一定是设备 1). 否则, 指定设备的显示名称和/或索引. 例如, 1, "Speakers", "Speakers:2" 或 "Speakers (Example HD Audio)".

有关详情, 请参阅 Device (Sound 函数).

错误处理

如果无法找到设备或组件, 或组件不支持此控制类型, 则抛出 TargetError. 否则失败时, 抛出 OSError.

备注

调整音量的另一种方法是让脚本发送音量控制按键来改变整个系统的主音量, 如下例所示:

Send "{Volume_Up}"  ; 增加 1 个音程的主音量(通常为 5%).
Send "{Volume_Down 3}"  ; 降低 3 个音程的主音量.

要发现系统上安装的声音设备的功能 -- 比如名称和可用的组件 -- 请运行这个声卡分析脚本.

SoundSetVolume 在更改音量级别时会尝试保持现有的平衡.

使用 SoundGetVolume 来获取当前音量设置.

Sound 函数

示例

将主音量设置为百分之 50. 引号可以省略.

SoundSetVolume "50"
SoundSetVolume 50

将主音量增加百分之 10. 引号 不能 省略.

SoundSetVolume "+10"

将主音量减小百分之 10. 引号可以省略.

SoundSetVolume "-10"
SoundSetVolume -10

将麦克风录音音量增加百分之 20.

SoundSetVolume "+20", , "Microphone"
