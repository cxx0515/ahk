EnvSet

将值写入指定的环境变量中.

EnvSet EnvVar , Value

参数

EnvVar

类型: 字符串

环境变量的名称, 例如 "Path".

Value

类型: 字符串

如果省略, 则删除环境变量. 否则, 请指定要写入的值.

错误处理

失败时抛出 OSError.

备注

操作系统限制每个环境变量的文本最多为 32 KB.

此函数创建或改变的环境变量只有由此脚本通过 Run 或 RunWait 运行的程序才可以访问. 有关详情, 请参阅环境变量.

此函数的存在是因为普通脚本变量没有存储在环境变量中. 这是因为性能会变差, 也因为操作系统将环境变量限制为 32 KB.

相关

EnvGet, Run / RunWait

示例

将一些文字写入环境变量中.

EnvSet "AutGUI", "Some text to put in the environment variable."
