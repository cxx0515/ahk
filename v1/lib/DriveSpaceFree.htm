DriveSpaceFree

包含指定路径的驱动器的可用磁盘空间, 单位为 MB.

DriveSpaceFree, OutputVar, Path

参数

OutputVar

用来保存结果的输出变量, 结果向下取整到最近的整数.

Path

驱动器包含的任何路径(可能也适用于 UNC 路径和映射驱动器).

备注

OutputVar 被设置为可用的磁盘空间数, 单位为 MB(向下取整到最近的兆字节).

一般来说, Path 可以是任何路径. 由于 NTFS 支持挂载的卷和目录连接, 具有相同驱动器号的不同路径可以产生不同数量的空闲空间.

相关

Drive, DriveGet

示例

检索 C 盘的可用磁盘空间, 并将其存储在 FreeSpace 中.

DriveSpaceFree, FreeSpace, c:\

检索并报告包含 A_MyDocuments 的驱动器的空闲磁盘空间.

DriveSpaceFree, FreeSpace, % A_MyDocuments
MsgBox % FreeSpace " MB"
